## Expected Outcomes

Running the monitoring code will:
- Emit structured JSON metrics to stdout at configurable intervals
- Track latency (p50/p99), error rates, and throughput
- Alert when p99 latency exceeds 500ms or error rate > 1%
- Continue running until interrupted (Ctrl+C)
